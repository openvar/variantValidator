# VariantValidator Web Interface (VVweb)

Welcome to the VariantValidator Web Interface (VVweb) documentation.

VVweb provides an easy-to-use platform for validating, mapping, formatting and exploring human sequence variant descriptions without requiring software installation or programming knowledge.

Whether you are validating a single variant, analysing a list of variants, or identifying suitable transcript reference sequences for a gene, VVweb provides tools to support research, clinical and diagnostic workflows.".

---

# Getting Started

Choose the VVweb tool that best matches your task.

## Validate a single variant

The **Validator** allows you to validate a single sequence variant description interactively.

Use this tool to:

- Validate HGVS variant descriptions.
- Correct common formatting errors.
- Map variants between transcript, genomic and protein reference sequences.
- Explore detailed validation results.

→ [Validator](validator.md)

---

## Validate multiple variants

The **Batch Validator** validates multiple variant descriptions in a single submission.

Use this tool when:

- validating large variant lists;
- processing sequencing results;
- preparing supplementary data for publication; or
- validating variants prior to journal submission.

Results are processed asynchronously and returned by email.

→ [Batch Validator](batch_validator.md)

---

## Find transcript reference sequences

The **Gene2Transcripts** tool helps identify transcript reference sequences associated with a gene.

Use this tool to:

- discover RefSeq and Ensembl transcripts;
- identify MANE transcripts;
- obtain transcript metadata; and
- select appropriate transcripts before validating variants.

→ [Gene2Transcripts](gene2transcripts.md)
