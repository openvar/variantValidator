# REST VariantValidator2 (SHAIP)

REST VariantValidator2 (SHAIP) is the current-generation REST API for VariantValidator.

Built using FastAPI and Python 3.12, REST VariantValidator2 provides a modern, high-performance interface to the VariantValidator framework, supporting automated variant validation, formatting, transcript mapping and scalable bioinformatics workflows.

REST VariantValidator2 is intended for production deployments, cloud-native applications and high-throughput pipeline integration.

---

# Repository access

The REST VariantValidator2 source repository is **private** and **will not be made publicly accessible**.

Users interact with REST VariantValidator2 through the hosted service and the official documentation rather than through the source repository.

---

# Requesting access

Access to the hosted REST VariantValidator2 service requires a user account.

Instructions for requesting an account, activating access and obtaining API credentials are provided in the official REST VariantValidator2 documentation.

If you require assistance with account activation or access requests, please contact the VariantValidator development team:

**admin@variantvalidator.org**
